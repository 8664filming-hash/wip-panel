▸ 使用說明
🔄
Run — 讀取最新 WIP 工作表，AI 自動解析所有含 @ 的任務並交叉比對行事曆。每次更新 WIP 後按一次即可。
👤
人員篩選 — Run 完後點選姓名，即時篩選該人員相關任務，不需重新讀取。
✓
打勾 — 確認這筆要處理。打勾後按底部「新增至好咖行事曆」才會真的新增或更新。沒打勾的不會被處理。
✏
鉛筆 — 展開行事曆選取器，手動選取對應的行事曆事件（當 AI 比對錯誤時使用）。也可點「❌ 此筆確定沒有」標記為待新增。
✕
叉叉 — 忽略這筆，不列入處理。可再按一次取消忽略。
🟡
顏色選單 — 選擇行事曆事件顏色。預設「❓ 保留」代表沿用行事曆原本顏色（更新時不覆蓋），新建時預設黃色。
✨
給 Gemini 下指令 — 輸入自然語言指令直接修改清單，例如「把奶茶弟讀本改成 3/20 下午三點」或「幫小點點新增 3/24 空拍 @Borry 石頭」。
✓ 已建 / 待新增
行事曆狀態 — AI 判斷行事曆是否已有對應事件。判斷錯誤時用鉛筆手動修正。
🟡 新增至好咖行事曆
處理所有打勾的項目：✅已建 + 有對應事件 → 更新原事件；❌待新增 → 新建事件。
📱 新增至 Mac 提醒
將打勾項目送到 Mac 提醒事項，需安裝「好咖提醒事項」捷徑。

按 Run 載入資料

✨ 給 Gemini 下指令

用 ✏ 在各項目寫修改指令，或在下方輸入整體指令（加事件、改標題、調整日期…），送出後 Gemini 直接更新清單。
